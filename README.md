# Java Game Backend Simulator

This project is a backend Java service that simulates core systems behind an online game, including player management, match sessions, and leaderboards.

The goal of this project is to demonstrate backend system design, object-oriented programming, and production-style Java development.

## Planned Features
- Player creation and management
- Match lifecycle (start, end, tracking)
- Leaderboard and scoring system
- Data persistence
- REST API (Spring Boot)

## Tech Stack
- Java
- Spring Boot (planned)
- SQL database (planned)
